# 错误解决方案

## 错误信息

### 1. 后端连接错误
```
GET http://localhost:5000/DappCommon/GetChainNetworkConfigs 
net::ERR_CONNECTION_REFUSED
```

**原因**: 后端服务未运行

**解决方案**:
1. 启动后端服务
2. 等待服务完全启动（约 15-30 秒）
3. 刷新前端页面

---

### 2. Bitget Wallet 扩展错误
```
POST https://infragrid.v.network/wallet/getnodeinfo 
net::ERR_CONNECTION_CLOSED
```

**原因**: Bitget Wallet 浏览器扩展尝试连接外部服务失败

**说明**: 这是钱包扩展的内部请求，不影响主要功能，可以忽略

## 常见问题

### Q: 后端服务启动失败
**A**: 检查:
1. 数据库服务是否运行
2. 数据库连接字符串是否正确
3. 端口 5000 是否被占用
4. 查看后端窗口的错误信息

### Q: 前端仍然无法连接
**A**: 
1. 确认后端服务已完全启动（等待 15-30 秒）
2. 检查浏览器控制台是否有其他错误
3. 清除浏览器缓存并刷新
4. 检查前端 `.env` 文件中的 API 地址配置

### Q: 端口被占用
**A**: 
1. 查找占用端口的进程:
   ```cmd
   netstat -ano | findstr :5000
   ```
2. 结束进程或更改后端端口配置
