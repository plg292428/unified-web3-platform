## 🎯 核心业务流程

### 电商流程
1. 浏览商品 → 搜索/筛选 → 查看详情
2. 加入购物车 → 修改数量 → 结算
3. 创建订单 → 选择支付方式 → 支付 → 确认支付
4. 订单发货 → 物流跟踪 → 订单完成
5. 商品评价 → 商家回复 → 评价投票

### 资产管理流程
1. 钱包连接 → 授权代币 → 转账到链上（充值）
2. 查看资产 → 转账到钱包（提现）→ 订单跟踪
3. 查看历史记录 → 资产统计

### AI交易流程
1. 激活AI交易 → 创建交易订单 → 等待完成 → 查看奖励

### 挖矿流程
1. 查看挖矿状态 → 查看奖励记录 → 资产统计
