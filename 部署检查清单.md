# 部署检查清单

**使用说明**: 在部署前和部署后，请逐项检查以下内容。

---

## 📋 部署前检查

### 环境准备
- [ ] .NET SDK 8.0+ 已安装
- [ ] Node.js 18+ 已安装
- [ ] SQL Server 已安装并运行
- [ ] 数据库已创建
- [ ] 数据库迁移已执行

### 后端检查
- [ ] `appsettings.json` 配置正确
- [ ] 数据库连接字符串正确
- [ ] CORS 配置包含前端域名
- [ ] 后端服务可正常启动
- [ ] Swagger 可正常访问

### 前端检查
- [ ] `.env` 或 `.env.production` 配置正确
- [ ] `VITE_API_BASE_URL` 指向正确的后端地址
- [ ] 前端可正常构建 (`npm run build`)
- [ ] 构建产物在 `dist/` 目录

### 功能测试
- [ ] API 测试通过 (`test_api.ps1`)
- [ ] 数据库测试通过 (`test_database.ps1`)
- [ ] 本地功能测试通过

---

## 📋 部署后检查

### 前端部署检查
- [ ] 前端页面可正常访问
- [ ] 页面无 JavaScript 错误
- [ ] API 请求可正常发送
- [ ] 钱包连接功能正常

### 后端部署检查
- [ ] 后端 API 可正常访问
- [ ] Swagger 可正常访问（如果启用）
- [ ] 数据库连接正常
- [ ] 定时任务正常运行

### 功能验证
- [ ] 商品列表可正常加载
- [ ] 商品搜索功能正常
- [ ] 购物车功能正常
- [ ] 订单创建流程正常
- [ ] Web3 支付流程正常
- [ ] 订单状态更新正常

### 性能检查
- [ ] 页面加载速度正常
- [ ] API 响应时间正常
- [ ] 无内存泄漏
- [ ] 无控制台错误

- [ ] CORS 配置正确
- [ ] HTTPS 证书正常
- [ ] 无 522/503 错误
